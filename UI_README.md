# Kivy UI для RPG игры

## Установка

1. Установите зависимости:
```bash
pip install -r requirements.txt
```

## Запуск

### Графический интерфейс (Kivy):
```bash
python main.py
```

### Консольный режим (старый интерфейс):
```bash
python main.py --console
```

## Возможности UI

### Главное меню
- 🆕 Новая игра
- 💾 Загрузить игру
- ← Выход

### Создание персонажа
- Ввод имени
- Выбор класса (Воин, Маг, Лучник)

### Игровой экран
- **Интерактивная карта** с тремя локациями:
  - 🌲 Лес
  - ⛏️ Пещера
  - 🏛️ Руины
- **Статистика игрока** (HP, уровень, монеты, XP)
- **Меню действий**:
  - 🏰 Таверна (квесты и спутники)
  - 🛒 Магазин (покупка/продажа)
  - 🎰 Казино (игры)
  - 🎒 Инвентарь (экипировка)
  - 📊 Статус (подробная информация)
  - 💾 Сохранить игру

### Экран боя
- Отображение врагов с HP
- Лог боевых действий
- Кнопки действий:
  - ⚔️ Атаковать
  - 🧪 Использовать зелье
  - 🏃 Убежать
  - 🏳️ Сдаться

### Другие экраны
- **Таверна**: просмотр квестов, найм спутников
- **Магазин**: покупка и продажа предметов
- **Казино**: игры (Орёл/Решка, Слоты, Колесо Фортуны)
- **Инвентарь**: управление предметами и экипировкой
- **Статус**: подробная статистика персонажа

## Особенности

- Полностью графический интерфейс
- Интерактивная карта для выбора локаций
- Удобные кнопки для всех действий
- Поддержка сохранений
- Адаптивный дизайн для разных размеров экрана

## Для Android

Для создания APK для Android используйте Buildozer:

1. Установите Buildozer
2. Создайте `buildozer.spec` файл
3. Запустите `buildozer android debug`
