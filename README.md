# 🎮 Mini RPG - Улучшенная версия

Полнофункциональная RPG игра на Python с современной архитектурой, системой сохранений, боевой системой и множеством контента.

## 🎯 Основные возможности

### ✨ Новые функции:
- **💾 Система сохранений** - сохраняйте прогресс и продолжайте игру
- **💀 Система смерти** - при смерти показывается полная статистика сессии
- **🎭 Модульная архитектура** - код разбит на логические модули
- **📚 Больше контента**:
  - 3 локации: Лес, Пещеры, Руины
  - Расширенная система врагов
  - Больше квестов и вознаграждений
  - Больше предметов и экипировки
- **🎮 Улучшенный UX**:
  - Красивый интерфейс с эмодзи
  - Подробная статистика
  - События в локациях

## 📁 Структура проекта

```
NLH_remake/
├── main.py              # Точка входа
├── game.py             # Главный модуль игры и интерфейс
├── creatures.py        # Персонажи, враги, спутники
├── items.py           # Система предметов и инвентаря
├── battle.py          # Боевая система и враги
├── quests.py          # Квесты и таверна
├── shop_casino.py     # Магазин и казино

├── utils.py           # Вспомогательные функции

```

## 🚀 Запуск

```bash
python main.py
```

## 🎮 Как играть

### Главное меню
1. **Новая игра** - начните с нуля
2. **Загрузить игру** - продолжите с сохранения
3. **Выход** - закройте игру

### Локации
- **🌲 Лес** - легкие враги, события
- **⛏️ Пещера** - средние враги, сокровища
- **🏛️ Руины** - сложные враги, легенды

### Меню
- **🏰 Таверна** - квесты и найм спутников
- **🛒 Магазин** - покупка и продажа предметов
- **🎰 Казино** - азартные игры
- **🎒 Инвентарь** - управление предметами
- **📊 Статус** - характеристики персонажа
- **💾 Сохранить** - сохранить прогресс

## 👥 Персонажи

### Классы
1. **Воин** - высокий HP, средний урон
2. **Маг** - высокий урон, низкий HP
3. **Лучник** - баланс характеристик

### Спутники
- **Танк** - высокая защита
- **Лучник** - высокий урон
- **Целитель** - восстановление HP

## 🗡️ Система боя

### Действия в бою:
1. **Атаковать** - нанесите урон врагу
2. **Использовать зелье** - восстановите HP
3. **Убежать (30%)** - попытка сбежать
4. **Сдаться** - конец игры

### Враги масштабируются по уровню
- Враги получают +12% к HP за каждый уровень
- Враги получают +5% к урону за каждый уровень
- Награды растут вместе с уровнем

## 📈 Система опыта и уровней

- Получайте опыт от победы над врагами
- Получайте опыт от квестов
- При повышении уровня восстанавливается HP
- Урон увеличивается на 5% за уровень

## 🎁 Предметы

### Рарность
- ⚪ **Common** (50%) - базовые предметы
- 🟢 **Uncommon** (25%) - редкие предметы
- 🔵 **Rare** (12%) - очень редкие
- 🟣 **Epic** (8%) - эпические
- 🟡 **Legendary** (5%) - легендарные

### Типы
- **⚔️ Оружие** - увеличивает урон
- **🛡️ Броня** - увеличивает защиту
- **🧪 Зелья** - восстанавливают HP

## 📊 Статистика сессии

При смерти выводится:
- Финальный уровень
- Финальные монеты
- Общий выданный урон
- Общий полученный урон
- Количество побежденных врагов
- Количество проведённых битв
- Предметы в инвентаре

## 💾 Система сохранений

- Сохранения хранятся в папке `saves/` в формате JSON
- Можно иметь несколько сохранений
- При загрузке восстанавливается полное состояние игры:
  - Характеристики персонажа
  - Инвентарь
  - Экипировка
  - Спутники
  - Статистика

## 🎲 Казино

### Игры:
1. **Орёл/Решка** - 50% на выигрыш ставки
2. **Слоты** - шанс на джекпот до 10x
3. **Колесо Фортуны** - множители от 1x до 8x

## ⚡ События в локациях

### Лес:
- Сундук с предметом
- Засада
- Целебный источник
- Проклятая область

### Пещера:
- Клад (монеты)
- Ловушка (урон)
- Встреча с учёным (опыт)

## 📜 Квесты

Доступные квесты:
1. **Убей 3 волков** - 150 монет + 120 XP
2. **Избави лес от 5 гоблинов** - 200 монет + 150 XP
3. **Выиграй 3 битвы подряд** - 250 монет + 200 XP + Меч
4. **Исследуй пещеру (5 побед)** - 400 монет + 300 XP + Зелье
5. **Победи 10 врагов** - 1000 монет + 500 XP + Легендарный меч
