# NLH Remake - Mini RPG Game

**Современная RPG игра на Python с полноценной системой боя, квестов и сохранений.**

## 📋 Структура Проекта

```
NLH_remake/
├── main.py              # Точка входа (запуск игры)
├── core/                # Основная логика игры
│   ├── __init__.py
│   ├── game.py         # Главный класс Game
│   ├── creatures.py    # Player и Creature классы
│   ├── config.py       # Конфигурация и константы
│   └── utils.py        # Утилиты
├── data/                # Игровые данные
│   ├── __init__.py
│   ├── enemies.py      # Враги и боссы
│   ├── items.py        # Предметы и инвентарь
│   └── locations.py    # Локации и менеджер
├── systems/             # Игровые системы
│   ├── __init__.py
│   ├── battle.py       # Боевая система
│   ├── quests.py       # Квесты
│   ├── active_quests.py # Отслеживание квестов
│   ├── npcs.py         # NPC
│   ├── npc_system.py   # Система диалогов NPC
│   ├── shop_casino.py  # Магазин и казино
│   └── save_system.py  # Сохранения
├── ui/                  # User Interface
│   ├── __init__.py
│   └── ui_app.py       # Kivy UI приложение
├── tests/               # Тесты
│   ├── __init__.py
│   ├── test_full_playthrough.py
│   ├── test_ancient_cave.py
│   ├── test_ancient_cave_bosses.py
│   └── quick_shop_test.py
├── docs/                # Документация
└── saves/               # Сохранённые игры
```

## 🚀 Быстрый Старт

### Требования
- Python 3.8+
- Kivy 2.3.1
- Windows/Linux/Mac

### Установка зависимостей
```bash
pip install -r requirements.txt
```

### Запуск Игры

**Графический режим (Kivy UI):**
```bash
python main.py
```

**Консольный режим (текстовая игра):**
```bash
python main.py --console
```

### Запуск Тестов

**Все тесты:**
```bash
python -m pytest tests/ -v
```

**Конкретный тест:**
```bash
python tests/test_full_playthrough.py
python tests/test_ancient_cave.py
python tests/test_ancient_cave_bosses.py
```

## 🎮 Геймплей

### Локации
1. **Лес** (Forest) - Начальная локация
2. **Болото** (Swamp) - Открывается после леса
3. **Шахты** (Mines) - Открывается после болота
4. **Горы** (Mountains) - Открывается после шахт
5. **Пещера Древних** (Ancient Cave) - 4 прогрессивных босса

### Системы
- **Боевая система** - ATB-подобная система с расчётом урона и защиты
- **Система предметов** - 50+ предметов (оружие, броня, расходники)
- **Враги и боссы** - 40+ типов врагов и 4 уникальных босса
- **Система опыта** - Прогрессия уровней с прокачкой характеристик
- **NPC и квесты** - Диалоги с NPC и система квестов
- **Магазин и казино** - Торговля и азартные игры
- **Сохранения** - Полная система сохранения/загрузки

## 📦 Расширяемость

**Добавить нового врага:**
```python
# В data/enemies.py
EnemyDatabase.initialize()  # автоматически загружает всех врагов
```

**Добавить новый предмет:**
```python
# В data/items.py
ItemDatabase.initialize()  # загружает все предметы
```

**Добавить новую локацию:**
```python
# В data/locations.py
LocationManager.get_locations()  # получить все локации
```
