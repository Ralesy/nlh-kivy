## 🔄 Сравнение форматов показа информации о предметах

### ДО (минимальный формат)

```
┌─────────────────────────────┐
│   Информация о предмете     │
├─────────────────────────────┤
│ Железный меч                │
│ Обычное оружие              │
│ Цена: 100 монет            │
│                             │
│ Способность: Удар          │
│ +5 урона за удар           │
└─────────────────────────────┘
```

**Проблемы:**
- Без информации о материале
- Без информации о состоянии
- Минимальные детали о способности
- Нет прокрутки (может не поместиться информация)
- Нет кнопки закрытия в некоторых версиях

---

### ПОСЛЕ (полный формат, синхронизированный)

```
┌────────────────────────────────┐
│  📖 Информация о предмете      │
├────────────────────────────────┤
│ Железный меч                   │
│ ⚔️ Урон: 15                    │
│ Материал: iron                 │
│ Состояние: идеальное           │
│ Цена: 100 монет               │
│                                │
│ 📝 Обычное боевое оружие       │
│ из железа                      │
│                                │
│ Способность: Боевой клич      │
│ (passive)                      │
│ +5 урона за удар              │
│ Игнорирует 15% брони          │
│ Крит. множитель: x1.5         │
│                                │
│  [    Закрыть    ]             │
└────────────────────────────────┘
```

**Улучшения:**
✅ Полная информация о материале
✅ Состояние предмета
✅ Подробное описание способности
✅ ScrollView для длинного текста
✅ Кнопка закрытия
✅ Единообразный размер поп-апа (0.8, 0.7)
✅ Правильный перенос текста (dp(320))

### 🎯 Результат

Пользователь может теперь в любом месте игры (магазин, инвентарь, таверна, боевой инвентарь) нажать кнопку информации о предмете и получить:

1. **Базовую информацию** - имя, тип
2. **Характеристики** - урон/защита, материал, состояние
3. **Стоимость** - цена предмета
4. **Описание** - подробное описание
5. **Способность** - полная информация о способности с параметрами

Все это в одном красивом, прокручиваемом окне с единообразным дизайном!
