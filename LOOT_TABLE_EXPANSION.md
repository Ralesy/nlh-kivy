# Enemy Loot Table Expansion - Complete

## Summary
Expanded all enemy loot tables to include ALL weapon types, not just swords and axes. Now players will find variety in drops: swords, axes, spears, bows, daggersю

## Changes Made

### Iron Weapons (Forest Enemies)
- **Мародёр (Raider)**: sword, axe, dagger, bow (was: only sword)
- **Бандит (Bandit)**: sword, axe, dagger, bow (was: only axe)
- **Дезертир (Deserter)**: sword, axe, spear, bow, dagger (was: only sword)

### Goblin Weapons (Swamp Enemies)
- **Гоблин (Goblin)**: dagger, cleaver, spear, bow, kris (was: only cleaver and dagger)
- **Болотник (Bog Walker)**: spear, bow (NEW - was only potions)

### Orc Weapons (Mine Enemies)
- **Орк (Orc)**: sword, maul, spear, bow, dagger (was: only sword and maul)
- **Крысиный Король (Rat King)**: sword, maul, spear, bow, dagger (was: only sword)

### Dwarf Weapons (Mine Enemies)
- **Гном-драугр (Dwarven Draugr)**: sword, axe, spear, dagger (was: only axe)
- **Старый Камнежор (Old Stonehammer)**: sword, axe, spear, dagger (was: only hammer)

## Weapon Drop Rates
Each enemy now drops weapons with these probabilities:
- Primary weapon type: 0.35 (35%)
- Secondary weapon type: 0.35 (35%)
- Tertiary weapon type: 0.25 (25%)
- Quaternary weapon type: 0.20 (20%)
- Quinary weapon type: 0.15 (15%)
- Armor/potions: Maintained at existing rates

## Result
Instead of seeing mostly swords and axes, players will now encounter:
- 🗡️ Swords (Melee)
- 🪓 Axes (Heavy Melee)
- 🎯 Bows (Ranged)
- 🗡️ Spears (Pole Arms)
- 🗡️ Daggers (Light Melee)
- 💀 Maces/Mauls (Crushing)

Each weapon type has unique passive and active abilities tied to it!
