## 🔍 Чек-лист проверки

Для каждого экрана проверьте:

### Базовая информация
- [ ] Имя предмета видно
- [ ] Для оружия: показан урон
- [ ] Для брони: показана защита
- [ ] Для зелий: показано восстанавливаемое HP

### Детальная информация
- [ ] Материал указан (iron, wood, leather, elf и т.д.)
- [ ] Состояние видно (идеальное, хорошее и т.д.)
- [ ] Цена показана
- [ ] Описание присутствует

### Способность
- [ ] Если способность есть, показано её имя
- [ ] Тип способности отображен
- [ ] Параметры способности видны (если есть)
  - [ ] Урон за удар
  - [ ] Игнорирование брони
  - [ ] Критический множитель

### Дизайн и функциональность
- [ ] Поп-ап правильного размера (0.8, 0.7)
- [ ] Текст переносится правильно (width = dp(320))
- [ ] ScrollView работает для длинных текстов
- [ ] Кнопка "Закрыть" работает
- [ ] Дизайн единообразен во всех экранах
- [ ] Нет обрезания информации

## 🐛 Если что-то не так

### Проблема: Информация обрезана
**Решение:** Проверьте что используется ScrollView в методе show_item_info()

### Проблема: Способность не показывается
**Решение:** Проверьте что в предмете установлено свойство ability
```python
if hasattr(item, 'ability') and item.ability:
    # Способность должна отображаться
```

### Проблема: Поп-ап неправильного размера
**Решение:** Проверьте что используется size_hint=(0.8, 0.7)

### Проблема: Текст не переносится правильно
**Решение:** Проверьте что используется text_size=(dp(320), None)
