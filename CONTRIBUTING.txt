5. ✅ Обработка ошибок - предусмотрите исключения

=== ДОБАВЛЕНИЕ НОВОГО ВРАГА ===

Файл: battle.py → EnemyGenerator.POOLS

Пример:
```python
"dragon": {
    "id": "dragon",
    "name": "Дракон",
    "hp": 200,
    "dmg": 30,
    "coins": 500
}
```

Свойства:
- hp: базовое здоровье (масштабируется по уровню)
- dmg: базовый урон (масштабируется по уровню)
- coins: награда за убийство

Врагу не нужны ID или Name в items.py.

=== ДОБАВЛЕНИЕ НОВОГО КВЕСТА ===

Файл: quests.py → Tavern._initialize_quests()

Пример:
```python
q_new = Quest(
    "q_new_quest",
    "Найди магический кристалл",
    {"Лесной гоблин": 10, "Волк": 5},
    {"coins": 300, "exp": 200, "item": "w_sword"}
)
self.available_quests[q_new.id] = q_new
```

Структура:
- id: уникальный ID (начните с q_)
- desc: описание квеста
- goal: словарь целей (враги -> количество)
- reward: награда (coins, exp, item)

Враги должны быть созданы в battle.py.
Items должны быть в items.py.

Пример:

@staticmethod
def roulette(bet: int) -> Tuple[str, int]:
    """Русская рулетка - рискованно!"""
    if bet <= 0:
        return "Ставка > 0", 0
    
    # 50/50: либо выигрыш 2x, либо потеря
    if random.random() < 0.5:
        return f"Вы выжили! Выигрыш {bet * 2}", bet * 2
    else:
        return "Вы проиграли всё", -bet
```

elif ch == "4":

```

=== ДОБАВЛЕНИЕ СИСТЕМЫ СКИЛЛОВ ===

Создать новый файл: skills.py

```python
from creatures import Player

class Skill:
    def __init__(self, name: str, damage: int, cost: int):
        self.name = name
        self.damage = damage
        self.cost = cost  # манна или энергия
    
    def use(self, user: Player, target: Creature) -> str:
        if user.mana < self.cost:
            return "Недостаточно маны"
        user.mana -= self.cost
        dealt = target.take_damage(self.damage)
        return f"{self.name} нанёс {dealt} урона"

class PlayerSkills:
    def __init__(self, player: Player):
        self.player = player
        self.skills = []
    
    def learn_skill(self, skill: Skill):
        self.skills.append(skill)
```

Затем в game.py → battle_loop() добавить опцию "Скилл".

=== ДОБАВЛЕНИЕ СИСТЕМЫ ДОСТИЖЕНИЙ ===

Создать новый файл: achievements.py

```python
class Achievement:
    def __init__(self, id_: str, name: str, condition_fn):
        self.id = id_
        self.name = name
        self.condition = condition_fn
        self.unlocked = False

class AchievementTracker:
    def __init__(self, player):
        self.player = player
        self.achievements = {}
    
    def check_all(self):
        for ach in self.achievements.values():
            if not ach.unlocked and ach.condition(self.player):
                ach.unlocked = True
                print(f"🏆 Достижение: {ach.name}")
```

Вызывать check_all() после каждого события.
